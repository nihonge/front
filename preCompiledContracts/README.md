# 预编译合约模块
使用go编写，每个预编译合约需要完成两个接口：

**RequiredGas(input []byte) uint64**:计算调用该合约需要花费的gas

**Run(input []byte) ([]byte, error)**:合约的执行逻辑，返回结果与error对象
## 预编译合约和地址
| 合约地址     | 合约名称        | 合约作用                                                 |
|--------------|-----------------|----------------------------------------------------------|
| `0x01`       | `encrypt`      | 明文加密                             |
| `0x02`       | `decrypt`         | 密文解密                            |
| `0x03`       | `compute`      | 密态数据计算                         |
| `0x04`       | `keyGenerator`       |用户密钥生成                                     |
